# VibeCoding — Mermaid диаграммы

В этом репозитории содержатся 5 Mermaid-диаграмм, сгенерированных при помощи LLM-чата и сохранённых как артефакты.

**Автор / никнейм:** `egor_shalin`

## Список диаграмм
- **diagram_1 (Sequence):** Авторизация пользователя через OAuth — показывает шаги между пользователем, браузером, авторизационным сервером и бэкендом.
- **diagram_2 (Flowchart):** Рабочий процесс разработки (план → разработка → тест → релиз).
- **diagram_3 (ER):** Простая модель блога (Users, Posts, Comments).
- **diagram_4 (Class):** Классы клиентской библиотеки API (APIClient, Auth, Cache).
- **diagram_5 (Gantt):** Пример проектного плана (дизайн, разработка, тестирование).

## Как воспроизвести изображения
Код каждой диаграммы находится в `artifacts/diagram_N_code.md`. Чтобы получить PNG:

1. Откройте соответствующий `*_code.md` и скопируйте блок внутри ```mermaid
2. Вставьте код в https://mermaid.live/
3. В правом верхнем меню: `Actions -> Download PNG`
4. Сохраните файл как `artifacts/diagram_N.png` (заменив существующую заглушку).
